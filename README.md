# SortingAlgorithm
## Thực nghiệm thời gian chạy của những thuật toán sắp xếp
Đây là Repo của em khi làm báo cáo về "Thử nghiệm các thuật toán sắp xếp nội" cho môn Cấu trúc dữ liệu và giải thuật.  
Các bạn có thể xem so sánh thời gian chạy trong file Sorting Report.  
Nếu muốn coi về cách mình implement các thuật toán có thể coi trong Sorting_Algo trong đó gồm (Heap Sort, Quick Sort, Merge Sort và sort của numpy)  
Các test case đã được mình đính kèm trong đó input1 là một dãy số thực tăng dần, input2 dãy giảm dần.  
Các test case từ 3 đến 6 là các dãy số thực được mình sinh random.  
Các test case từ 7 đến 10 là các dãy số nguyên cũng được mình sinh random.  
Mình có để code sinh test trong file gen.py nếu các bạn muốn sinh thêm và đo thời gian trên chính máy của các bạn.
